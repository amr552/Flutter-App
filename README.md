# Crystal Call - Unique P2P WebRTC Experience

A premium, modern, and unique WebRTC Peer-to-Peer calling application.

## Key Features
- **WebRTC P2P**: Direct peer-to-peer audio/video communication.
- **Glassmorphic UI**: High-end design with dynamic animations and vibrant gradients.
- **Instant Hosting**: Single-file web application in the root `index.html` for easy GitHub Pages hosting.
- **Flutter Support**: Full Flutter implementation included in `lib/` for mobile development.
